# Template

boil-hello-world

Boilermaker Hello World in Python.

# Install

> If not done already, [install Boilermaker](https://boilermaker.dev/docs/install).

```bash
boil install https://github.com/yeajustmars/boil-hello-world --lang=python
```


# Usage

```bash
# boil install https://github.com/yeajustmars/boil-hello-world --lang=python

boil new boil-hello-world -l python -Od /tmp -n boil-py

cd /tmp/boil-py

python3 src/hello_world.py
```
